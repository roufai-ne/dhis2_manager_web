Description

Cette plateforme est conçue pour la DSI du MESRIT afin de faciliter la collecte, la transformation et l'intégration des données statistiques dans le système d'information national.

Elle permet aux gestionnaires de données de générer des templates de collecte standardisés, de valider les données saisies et de préparer les imports automatiques vers le serveur central DHIS2.

Fonctionnalités Principales

Configuration

  • Connexion API directe
  • Synchronisation des métadonnées

Générateur

  • Templates Excel standardisés
  • Filtrage par Groupe et Niveau

Calculateur

  • Validation des données
  • Génération de payload JSON
